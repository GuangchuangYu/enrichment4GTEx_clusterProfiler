
## GO (BP) enrichment with clusterProfiler v4

``` r
eg=bitr(gene, "ENSEMBL", "ENTREZID", "org.Hs.eg.db")[, "ENTREZID"]
clusterProfiler_bp4 <- enrichGO(eg, ont="BP", OrgDb = org.Hs.eg.db, maxGSSize = NULL)
```
